# SECOM_Semiconductor_Manufacturing_Condition_Monitoring

## 📊 Dataset Overview

The SECOM dataset contains sensor measurements from a semiconductor manufacturing process.

- 1567 samples, 591 features
- Binary target: Pass (-1) / Fail (1)
- Highly imbalanced (~6.6% failures)
- Contains missing values (NaN)

### 🎯 Goal
Identify key features affecting production yield and build predictive models for failure detection.

### ⚠️ Challenges
- High dimensionality (many irrelevant features)
- Missing values
- Noisy sensor data
- Severe class imbalance
